## 记第一次完整的做移动端页面UI

### 移动端页面制作知识储备
- flex 布局
- 负 margin 增加元素宽度
- 媒体查询 做响应式
- meta 标签 禁止缩放
- 移动端适配（动态REM自适应布局） 
- 移动端设备特点

### PC页面制作 与 移动页面制作不同

移动页面需要做成响应式(Responsive)页面, 比PC制作页面多了几个步骤：媒体查询 + meta 标签 + 移动端适配（动态REM）。并且**PC页面制作更多考虑浏览器的兼容性**，若要兼容IE, 不能使用 flex 布局、rem 和一些CSS3属性，只能使用 float 布局 + 负margin。
